# my_printf

**Lightweight C library providing printf-style string formatting with full format-specifier support.**

## Features

- Full support for standard printf format specifiers (e.g., %d, %s, %f, etc.)
- Custom format specifiers (e.g., %b for binary, %r for reversed string)
- Support for field width, precision, and flags (e.g., '-', '+', '0', etc.)
- Thread-safe and reentrant design
- Minimal dependencies, suitable for embedded systems and low-level programming
- Comprehensive test suite to ensure correctness and reliability
- Optional Doxygen documentation for easy reference
